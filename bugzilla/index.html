BOSS Bugzilla Scan

This page allows you to scan the Bugzilla trackers used by BOSSv2 and extract each plugin entry's description and comments. These are then outputted in the v3 masterlist syntax, for easy copy/pasting. It is recommended that the output is checked and edited to remove non-URL data.

Everything is escaped to be YAML-safe. The most obvious consequence is that messages with line breaks will have them replaced with \n.

If a date is supplied below, then only descriptions/comments added after this date will be extracted.
