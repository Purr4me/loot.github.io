BOSS Bugzilla Scan

This page extracts plugin description/comment data from BOSSv2's Bugzilla trackers, filters the data to remove junk and duplicates, and merges it with the latest masterlists as URL metadata. The merged masterlist is then linked to for download.

If a date is supplied below, then only descriptions/comments added after this date will be extracted. The extraction can also be filtered by the resolution value of each Bugzilla entry. It is recommended that the output is checked and edited to remove non-URL data.

This page was tested on Chrome only, though it should also be fully-functional on recent versions of Firefox and Opera.
